Redactor Backbone.BUI.Redactor

About

Redactor is an amazing WYSIWYG editor for JQUERY and it is the best example of what Backbone.BUI library can do. You can plug any external library into it thanks to its Backbone.BUI.Loader object.

Check Redactor Site for a complete list of configuration options for Redactor plugin.

Example Redactor

The code

/* nothing is easier than this! */
var redactor = new Backbone.BUI.Redactor(
{
    /* if we do not include a jquery object, it will be appended to the body tag */
    renderTo: this.$('#sample-redactor'),
    /* setting the height within the style of the element is a must according to Redactor to modify its default height */
    attributes:{'style':'height:250px'}
});

redactor.render();
